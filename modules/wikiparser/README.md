# Wikiparser module

This is a older markup language, which is similar to some common markup languages like Markdown.

This module is necessary for old content, which is based on this language.

## Syntax 

### Simple text
- \***bold text**\*
- \__italice text_\_
- =`fixed width`=

### Definition lists

some::thing

### Source text
=
```
source text
```
=

### Quotes

\>
> multiline quote

\>


### Headlines

\+ Headline 1\
\++ Headline 2\
\+++ Headline 3\
\++++ Headline 4\
\+++++ Headline 5\
\++++++ Headline 6\

Headline 1\
\==========

Headline 2\
\----------

Headline 3\
\..........


### List

\- some\
\- thing

### Numbered List

\# some\
\# thing


### Links

Link->http://...

### Tables

|!Headline||!Headline over 2 columns|!Headline\
|Text|Text|(cssclass)Text|Text
